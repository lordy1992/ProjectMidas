Project Midas

A software solution that allows you to control your computer with the Myo Armband, completely replacing the mouse and keyboard.

ProjectMidas

With Project Midas, you can completely control your computer with a combination of arm movements and hand gestures! Anything that can be done with a traditional mouse and keyboard combination can be done with Midas.

Please refer to the generated documentation (accessible by the link in the sidebar) for a technical overview.
