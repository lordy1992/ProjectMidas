Detailed Description

The MainGUI class is the parent GUI of all the widgets used in Midas. It contains the mouse indicator, sequence displayer, and info indicator. The mouse indicator shows the relative mouse velocity, the sequence displayer shows the gesture sequences, and the info indicator shows the current status of Midas.
